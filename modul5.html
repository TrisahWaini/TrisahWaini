Cara Menampilkan Data dalam JavaScript

1. Console.log()

Console.log() adalah metode paling dasar untuk menampilkan data dalam JavaScript. Metode ini berguna untuk debugging dan menampilkan data di console browser. Contoh penggunaan:

console.log("Hello World");
console.log({ name: "John", age: 30 });

2. innerHTML / textContent

Menggunakan innerHTML atau textContent untuk memanipulasi konten HTML secara langsung. innerHTML memungkinkan penambahan markup HTML, sementara textContent hanya untuk teks biasa. Contoh:

document.getElementById("demo").innerHTML = "Hello World";
element.textContent = "Text content here";

3. DOM Manipulation

Menciptakan dan memanipulasi elemen DOM secara dinamis menggunakan createElement() dan appendChild(). Metode ini memberikan kontrol penuh atas struktur dokumen. Contoh:

const div = document.createElement("div");
div.textContent = "New Element";
document.body.appendChild(div);

4. Alert dan Prompt

Menggunakan dialog browser bawaan untuk menampilkan atau meminta input data dari pengguna. Meskipun sederhana, metode ini efektif untuk interaksi cepat. Contoh:

alert("Pesan pemberitahuan");
const input = prompt("Masukkan nama:");
